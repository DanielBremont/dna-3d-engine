## How to deploy

1. Synthesise oligonucleotides from [cube3d.dna](./cube3d.dna) file.
2. Arrange test tubes as shown on the diagram below.

<details>
<summary>Environment variables</summary>

```
Q = 0.01
cxtm = 0.606
axp = 0.606
cytm = 0.898
ayp = 0.898
cztm = 1.243
azp = 1.243
mxyzm = 0.3
nx = 0.036 + 0.555 X + 0.147 Y
ny = 0.853 + -0.517 Y
nz = 0.737 + -0.270 X + 0.302 Y
```
</details>
